Una realidad superpuesta

Las realidades que vemos

La realidad aumentada es una realidad creada a partir de nuestra realidad para aportar información. 

Pero esta realidad no se sustituye a la nuestra si no que se sobrepone a esta para aportar información adicional a la que ya vemos a simple vista.

Para poder acceder a esta realidad necesitamos de un dispositivo adecuado para su visualización como un smartphone o una tablet en las cuales debe de estar instaladas las aplicaciones que permiten ver las diferentes realidades aumentadas.
